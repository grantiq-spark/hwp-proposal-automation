DOCX → HWP 변환 번들 빌더

DOCX 파일을 올리면 한 묶음 ZIP을 만들어 드립니다. ZIP을 본인 Windows PC에서 풀고 convert.vbs 를 더블클릭만 하면 됩니다. 검은 창 안 떠요. API 키 필요 없고, 어떤 데이터도 서버에 저장되지 않아요.

고급 옵션 (보통 그대로 두면 됨)
상태 대기
번들 크기 - 

          

        


        

          받은 ZIP 실행하기 — 클릭 3번이면 끝
          
    
            
  1. 다운로드 폴더의 hwp-runner-XXXXXX.zip 우클릭 → "압축 풀기"
    2. 
            
  2. 풀어진 폴더에 들어가서 convert.vbs 를 더블클릭
    3. 
            
  3. 다이얼로그 "변환할까요?" 가 뜨면 예 클릭. 그 다음부터는 마우스/키보드 건드리지 마세요.
    4. 
            
  4. 30초~1분 동안 Word와 한컴 한글이 자동으로 잠깐씩 떴다 사라집니다.
    5. 
            
  5. "완료! 폴더 열까요?" 다이얼로그가 뜨면 예 클릭 → 폴더에 final.hwpx 가 보입니다. 한글로 열어보세요.
    6. 
          

          
⚠️ 검은 터미널 창을 보고 싶지 않다면 convert.vbs 만 더블클릭하세요. run.cmd 는 디버깅용입니다.

        


        

          필수 설치 / 트러블슈팅
          
필수 (Windows PC):

          
    
            
  • Microsoft Word — 정품 또는 Office 365
    • 
            
  • 한컴 한글 2020 — 한 번은 직접 실행해 라이선스 통과시켜 두기
    • 
          

          
비교 리포트 옵션을 켰을 때만 추가:

          
    
            
  • ImageMagick
    • 
            
  • Ghostscript
    • 
          

          
문제가 생기면:

          
    
            
  • convert.vbs 가 빨간 에러 다이얼로그를 띄우면, 같은 폴더의 run.cmd 를 더블클릭해보세요. 검은 창에 자세한 에러 로그가 나옵니다.
    • 
            
  • "Word.Application not found" → Microsoft Word 재설치/수리
    • 
            
  • "Cannot create HwpCtrl" → 한컴 한글 2020을 한 번 직접 실행해서 라이선스 인증 통과
    • 
            
  • 한컴 한글 스크립트 보안 경고 → 한글 → 도구 → 환경설정 → 보안 → 스크립트 보안 → "보통" 또는 "낮음"
    • 
            
  • 변환된 페이지가 너무 길거나 짧음 → 고급 옵션의 Factor 값을 1.8~2.3 사이로 조정 후 다시 ZIP 받기
    •
